# telegram-password-saver
Simple telegram password saver with no much security
It captures a login and sends the credentials to your telegram bot

Example:
🔐 New login detected

URL: https://practicetestautomation.com/practice-test-login/
Username: student
Password: Password123


To use this extension you will need to create a telegram bot with @BotFather and to find your telegram ID using @getmyidx_bot
you can test the conection and enjoy your bot. Have fun!
